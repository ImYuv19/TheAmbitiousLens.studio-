## [1.0.0] - 2026-06-27

### Initial Public Release
*   **Hero Section:** Vertical scrolling carousel slideshow with spring dynamics.
*   **My Works (Portfolio):** Tabbed content categorization supporting main films, commercials, documentaries, and YouTube Shorts. Features a lazy-loaded iframe player.
*   **Motion Graphics:** 2x2 grid displaying motion designs with nocookie YouTube players and fallback card handles.
*   **Design Showcase:** Masonry poster concept gallery with keyboard-navigable fullscreen lightbox modal and scroll-lock integration.
*   **Creative Process:** Symmetrical infinity spline methodology path trail animation powered by requestAnimationFrame.
*   **About Section:** Viewfinder camera monitor mock styling containing pulsing REC indicators, running timecodes, discipline pills, and easter eggs.
*   **Contact Section:** Info blocks grid, custom budget brief calculators, and optimized email/WhatsApp pre-filled target mappings.
*   **Minimal Footer:** Transparent signature footer featuring bullet point credits, gradient divider lines, and Sanskrit personal signatures.
*   **Visual Continuity:** Smooth scroll Lenis integration, desaturated CSS grain overlays, and custom typography variables.
